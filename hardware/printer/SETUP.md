# Configuración de Impresora Epson L3210

## Conexión
- La Epson L3210 se conecta por USB a la PC de producción
- Usa tinta continua (Ecotank) - colores: CMYK
- Tamaño máximo: A4 (210x297mm)

## Drivers
1. Descargar driver desde: https://epson.com/support
2. Modelo: L3210 Series
3. Instalar con configuración predeterminada
4. La impresora aparece como cola CUPS/IPP en Windows

## Uso en el sistema
- El sistema NO controla la impresora directamente
- El admin ve la cola de trabajos en el panel web
- Abre el PDF asociado al trabajo
- Imprime manualmente con la Epson
- Marca "Impreso" en el sistema

## Papeles compatibles
| Tipo | Uso |
|------|-----|
| Vinilo adhesivo mate | Stickers, etiquetas |
| Vinilo adhesivo brillo | Stickers coloridos |
| Papel fotográfico | Fotos, tarjetas |
| Papel adhesivo | Etiquetas genéricas |
| Papel común | Pruebas, borradores |

## Mantenimiento
- Cabezal: limpiar cada 30 días o si hay rayas
- Tinta: revisar nivel semanalmente
- Alineación: después de cambiar tinta
